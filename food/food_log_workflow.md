# 🍽️ 食事ログ＆栄養管理ワークフロー

## 概要
ゆーちゃ・まっちゃの食事ログを週2回（土・日）チェックし、栄養バランスを振り返る。
日本食品標準成分表（八訂増補2023年版）をベースに分析する。

---

## トリガー
- 毎週土曜 06:00
- 毎週日曜 06:00

---

## ステップ1：食事ログの共有

Claudeに以下のように伝える。

```
今日の食事ログです。
【ゆーちゃ】
朝：
昼：
夜：
間食：

【まっちゃ】
朝：
昼：
夜：
間食：
```

---

## ステップ2：Claudeによる栄養分析

Claudeが八訂データベースをもとに以下を算出する。

| 項目 | ゆーちゃ | まっちゃ |
|-----|---------|---------|
| エネルギー（kcal） | - | - |
| たんぱく質（g） | - | - |
| 脂質（g） | - | - |
| 炭水化物（g） | - | - |
| 食塩相当量（g） | - | - |

---

## ステップ3：週間サマリー（土曜のみ）

土曜日は1週間分のサマリーをまとめる。

```
先週の食事ログをまとめて週間サマリーを出して。
```

- 平均摂取カロリー
- 栄養バランスの過不足
- 来週への改善アドバイス

---

## 注意事項

- 八訂（日本食品標準成分表2020年版増補2023年）に基づいて計算
- 外食・コンビニ食は近似値で対応
- 体重・体調の変化があれば合わせて共有するとアドバイス精度が上がる
