# 🍽️ 食事ログ＆栄養管理ワークフロー

## 概要
ゆーちゃ・まっちゃの食事ログを週2回（土・日）チェックし、栄養バランスを振り返る。
日本食品標準成分表（八訂増補2023年版）をベースに分析する。

---

## トリガー
- 毎週土曜 12:00
- 毎週日曜 12:00

---

## ステップ1：食材ボードを写真撮影

1. 冷蔵庫の食材ボードを撮影する
2. Claudeに画像を添付して送る（PCのClaudeデスクトップから）
3. 必要に応じて食材の訂正を伝える（例：「納豆ではなく鶏むね皮なし」）

---

## ステップ2：Claudeによる読み取り・栄養分析

Claudeが以下を行う。

1. 画像から食材名・グラム数を読み取り
2. 八訂データベースをもとに栄養素を推計
3. 以下のテーブルで出力

| 栄養素 | ゆーちゃ | まっちゃ |
|--------|---------|---------|
| エネルギー（kcal） | - | - |
| たんぱく質（g） | - | - |
| 脂質（g） | - | - |
| 炭水化物（g） | - | - |
| 食物繊維（g） | - | - |
| 食塩相当量（g） | - | - |

---

## ステップ3：CSVログをGitHubに保存（PCのみ・土曜推奨）

Claudeが2つのCSVを生成してGitHubにプッシュする。

### ファイル構成
```
daily-tasks/
└── food/
    └── logs/
        ├── ingredients.csv   # 食材ログ（日付・人別・食材・グラム数）
        └── nutrition.csv     # 栄養ログ（日付・人別・各栄養素）
```

### CSVフォーマット

**ingredients.csv**
```
date,person,ingredient,amount_g
2026-04-18,ゆーちゃ,キャベツ,171
```

**nutrition.csv**
```
date,person,energy_kcal,protein_g,fat_g,carb_g,fiber_g,salt_g
2026-04-18,ゆーちゃ,2750,115,75,365,22,3.5
```

### GitHubへの追記方法
- **PCのClaudeデスクトップからアクセス時**：ClaudeがGitHub MCPで直接プッシュ
- **モバイルからアクセス時**：次回PC作業時にまとめてプッシュ

---

## ステップ4：週間サマリー（土曜のみ）

土曜日は1週間分のサマリーをまとめる。

```
先週の食事ログをまとめて週間サマリーを出して。
```

- 平均摂取カロリー
- 栄養バランスの過不足
- 来週への改善アドバイス

---

## 注意事項

- 八訂（日本食品標準成分表2020年版増補2023年）に基づいて計算
- 外食・コンビニ食は近似値で対応
- 体重・体調の変化があれば合わせて共有するとアドバイス精度が上がる
- **GitHubへのプッシュはPCのClaudeデスクトップからのみ可能**
