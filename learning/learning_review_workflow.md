# 📖 週次学習振り返りワークフロー

## 概要
毎週月曜と土曜の朝6時に学習の振り返りをClaudeと一緒に行う。
松尾研講座・MCP学習・Kaggleハッカソン・論文読みなど幅広く対象とする。

---

## トリガー
- 毎週月曜 06:00（先週の振り返り）
- 毎週土曜 06:00（週末の深掘り・来週の計画）

---

## ステップ1：今週やったことを共有

Claudeに以下のように伝える。

```
今週の学習振り返りお願いします。
【やったこと】
-
-
【できなかったこと】
-
【気になったこと・疑問】
-
```

---

## ステップ2：Claudeによる振り返りサポート

Claudeが以下を行う。

1. **学習内容の整理**：やったことを体系的に整理
2. **理解度チェック**：疑問点に回答・補足説明
3. **つながりの発見**：講座・MCP・Kaggle・論文間のつながりを指摘
4. **来週の優先タスク提案**：締め切り・イベントを考慮して提案

---

## ステップ3：アウトプット確認（土曜のみ）

土曜は以下のアウトプット状況も確認する。

```
今週のアウトプット状況を確認したいです。
- X（学習ログ）：投稿した / しなかった
- Zenn（LLM競技write-up）：書いた / 書かなかった
- note（エッセイ）：書いた / 書かなかった
```
