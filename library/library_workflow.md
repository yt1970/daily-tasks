# 📚 図書館 返却＆延長ワークフロー

## 概要
北区図書館の借出リストを毎週土曜朝6時にチェックし、返却・延長を管理する。

---

## ステップ1：借出リストのコピペ

1. ブラウザで北区図書館マイページを開く
   - URL: https://www.library.city.kita.lg.jp/opw/OPW/OPWUSERINFO.CSP
2. 借出中リストの表をすべて選択してコピー
3. Claudeに貼り付けて以下のように指示する

```
以下が今日時点の借出リストです。今週土曜日のアクションをまとめてください。
（リストをここに貼り付け）
```

---

## ステップ2：Claudeによる分類

Claudeが以下の4カテゴリに自動分類する。

| カテゴリ | 条件 |
|---------|------|
| 🔴 返却必須 | 延長済み、または予約待ちあり・延長不可で期限が今週土曜 |
| 🟡 本日オンライン延長 | 延長可能で期限が今週土曜 → +2週間延長 |
| 🔵 来週以降に返却 | 期限が来週以降・延長不可 or 延長済み |
| 🟢 来週以降に延長 | 期限が来週以降・延長可能 |

---

## ステップ3：オンライン延長の実施

1. 北区図書館マイページを開く
   - URL: https://www.library.city.kita.lg.jp/opw/OPW/OPWUSERINFO.CSP
2. 🟡 リストの本を1冊ずつ延長手続きする
3. 延長後の返却期限日を確認する（通常＋14日）

---

## ステップ4：Googleカレンダーの更新（任意）

Claudeに以下のように指示する。

```
カレンダーの図書館タスクの説明欄を今日の結果に更新して。
```

Claudeが「📚 図書館 返却＆延長チェック」イベントの説明欄を最新の返却・延長リストに更新する。

---

## 注意事項

- **延長は返却期限日当日までオンラインで可能**（+14日）
- **延長済みの本は再延長不可**（必ず返却）
- **予約待ちのある本は延長不可**（必ず返却）
- 毎週土曜朝6時にGoogleカレンダーのリマインダーあり

---

## 参考：判定ロジック

```
延長済み → 返却必須
予約待ちあり AND 延長不可 → 返却必須
延長可能 AND 期限=今週土曜 → 本日延長
延長可能 AND 期限>今週土曜 → 来週以降に延長
延長不可（予約待ち）AND 期限>今週土曜 → 来週以降に返却
```
