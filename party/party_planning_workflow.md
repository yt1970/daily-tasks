# 🍻 四半期宴会幹事ワークフロー

## 概要
東京エリアの定期飲み会（四半期1回）を開催8週前からClaudeと一緒に段取りを進める。

---

## トリガー
- 開催予定日の8週前の土曜 06:00

---

## タイムライン

### 8週前：キックオフ
```
四半期宴会の幹事タスクをスタートします。
開催希望時期：
参加予定人数：
エリア：
予算感（一人あたり）：
```

Claudeが以下を行う。
- 日程候補の洗い出しサポート
- お店の条件整理

---

### 7週前：日程調整
- 参加者へ日程候補を送付（調整さんなど）
- Claudeが案内文を作成

```
日程調整の案内文を作って。参加者は〇〇人くらい。
```

---

### 6週前：お店候補リストアップ
```
〇〇エリアで〇〇人、予算〇〇円のお店候補を出して。
条件：個室 / 飲み放題 / ジャンル〇〇
```

Claudeがお店候補をリストアップ・地図表示する。

---

### 5週前：日程確定・お店予約
- 日程調整結果をもとに日程確定
- お店に予約を入れる
- Claudeがお店への予約連絡文を作成

```
お店への予約メールを作って。
店名：、日時：、人数：、コース：
```

---

### 4週前：参加者への案内
Claudeが参加者向け案内文を作成。

```
参加者への案内文を作って。
日時：、場所：、集合場所：、会費目安：
```

---

### 2週前：人数確認・お店に最終連絡
- 参加人数を確定してお店に連絡
- Claudeが最終確認メッセージを作成

---

### 前日：リマインド
Claudeが参加者向けリマインドメッセージを作成。

```
明日の宴会のリマインドメッセージを作って。
```

---

## 注意事項

- 四半期ごと（1月・4月・7月・10月目安）に開催
- 東京エリア・参加者は固定メンバー
- 幹事は持ち回りの場合あり
